# BaksDev Yandex Support

[![Version](https://img.shields.io/badge/version-7.1.0-blue)](https://github.com/baks-dev/yandex-support/releases)
![php 8.3+](https://img.shields.io/badge/php-min%208.3-red.svg)

Модуль техподдержки Yandex

## Установка

``` bash
$ composer require baks-dev/yandex-support
```

Тесты

``` bash
$ php bin/phpunit --group=yandex-support
```
